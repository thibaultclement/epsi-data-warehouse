\section*{Objectif}

Formaliser l’architecture actuelle et proposer une vision cible.

\subsection*{Travail demandé}

\begin{enumerate}
  \item Réaliser un schéma d’architecture globale du système :
  \begin{itemize}
    \item sources,
    \item ingestion,
    \item dbt,
    \item base SQLite,
    \item outil BI.
  \end{itemize}
  \item Identifier 3 limites actuelles du système.
  \item Proposer 3 améliorations possibles.
\end{enumerate}

\subsection*{Livrable}

\begin{itemize}
  \item Diagramme d’architecture (draw.io ou équivalent).
  \item Analyse rédigée (10 lignes).
\end{itemize}

Créer au minimum 4 endpoints :

\begin{itemize}

\end{itemize}

\textbf{2. Dockerisation}

\begin{itemize}
  \item Créer un \file{Dockerfile}.

\begin{itemize}
