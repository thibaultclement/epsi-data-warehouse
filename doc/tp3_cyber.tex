\section*{Parcours CYBER}

\subsection*{Objectif}

Sécuriser et gouverner le datawarehouse.

\subsection*{Travail demandé}

\textbf{1. Analyse de risques}

\begin{itemize}
  \item Identifier les données sensibles.
  \item Identifier les risques (accès non autorisé, inférence).
\end{itemize}

\textbf{2. Mise en place de contrôles}

\begin{itemize}
  \item Proposer 3 profils utilisateurs :
  \begin{itemize}
    \item administrateur,
    \item analyste,
    \item direction.
  \end{itemize}
  \item Mettre en place des vues restreintes.
  \item Implémenter une anonymisation de \file{patient\_id}.
\end{itemize}

\textbf{3. Gouvernance}

\begin{itemize}
  \item Proposer une politique de conservation des données.
  \item Expliquer comment répondre à une demande de suppression (RGPD).
\end{itemize}

\subsection*{Livrables}

\begin{itemize}
  \item Document d’analyse de risques.
  \item Implémentation SQL (vues sécurisées).
  \item Justification des choix.
\end{itemize}
